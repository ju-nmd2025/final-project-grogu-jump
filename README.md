Welcome to Grogu Jump!!

Help The Kid, named Grogu, to get back up to his home planet since he got lost.

This is a simple Doodle-Jump-like game built with p5canvas. You help Grogu jump upwards by landing on 
different platforms while the world scrolls down.

Gameplay
- Grogu jumps automatically when landing on the ground or on platforms
- Platforms scroll down as the player reaches higher positions
- The game starts from a start screen and begins when pressing the Space Key

Controls
- Left Arrow - Move Left
- Right Arrow - Move Right
- Space - Start the Game
- R key - Restart the game when won/lost

Built With
- JavaScript
- p5.js
- HTML / CSS

Project Structure
- `index.html` - Main HTML file
- `game.js` - Main game loop and logic
- `player.js` - Player class and movement
- `platform.js` - Platform class
- `assets/` - Images in game
